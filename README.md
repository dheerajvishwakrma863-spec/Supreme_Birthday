# 🎉 Digital Birthday Keepsake

A futuristic, interactive **Single-Page Application (SPA)** designed to turn a simple birthday wish into a **cinematic digital experience**.

This project blends **3D visuals, voice interaction, gamification, and audio-reactive UI** to create a memorable keepsake.

---

## ✨ Features

### 🔐 1. The Vault (Entry Experience)

* Glassmorphism login interface
* User enters name + uploads photos
* “Hold-to-Unlock” button with progress animation

---

### 🌌 2. 3D Interactive Environment

* Built using **Three.js**
* Floating 3D crystals (icosahedrons)
* Smooth animations with immersive depth

---

### 🎮 3. Gamified Experience

* User must break **8 floating crystals**
* Each interaction triggers:

  * Screen shake effect
  * Real-time counter update

---

### 🎤 4. Voice Command Engine (AI Interaction)

Powered by **Web Speech API**

Supported commands:

* “Play Music”
* “Pause Music”
* “Volume Up / Down”
* “Change Vibe” (dynamic background change)
* “Surprise” (triggers finale)

---

### 🎧 5. Audio Experience

* Background music: `birthday_song.mp3`
* Continuous playback with control via voice

---

### 🎉 6. Grand Finale (Keepsake Card)

* Glassmorphism floating card
* Personalized birthday message
* Smooth GSAP animations
* Surprise-triggered reveal

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Glassmorphism + Animations)**
* **JavaScript (Vanilla)**
* **Three.js** → 3D graphics
* **GSAP** → Smooth transitions
* **Web Speech API** → Voice commands
* **Web Audio API (Planned/Extendable)**

---

## 📱 Responsiveness

* Fully responsive design
* Works on desktop + mobile browsers

---

## 🚀 How to Run

1. Download or clone the repository
2. Make sure the audio file is present:

   ```bash
   birthday_song.mp3
   ```
3. Open `index.html` in your browser
